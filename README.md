# Jornada de Estudos: Python

Este repositório documenta minha evolução na linguagem Python, organizando o aprendizado entre conceitos teóricos, aplicações práticas e machine learning.

## 📂 Estrutura do Repositório

### `00-referencias/`
Material de apoio versionado para consulta em qualquer máquina:
- **30-days-of-python/** — curso completo com 30 dias de conteúdo (introdução, estruturas de dados, funções, POO, web scraping, APIs, Pandas e mais)
- **python_zero_POO.pdf** — livro de referência sobre orientação a objetos em Python

### `01-teoria/`
Notebooks Jupyter com anotações e testes de conceitos, bibliotecas e sintaxes:
- `01-introducao.ipynb` — fundamentos da linguagem
- `02-estrutura-de-dados.ipynb` — listas, tuplas, sets, dicionários
- `03-algoritmos.ipynb` — lógica e algoritmos
- `04-days.ipynb` — acompanhamento do 30 Days of Python

### `02-pratica/`
Projetos e exercícios desenvolvidos a partir de desafios e livros técnicos:
| Arquivo/Pasta | Descrição |
|---|---|
| `00-Desafios.ipynb` | Caderno de desafios de lógica |
| `01-churrasco.py` | Cálculo de churrasco |
| `02-gerenciador_de_tarefas.py` | Gerenciador de tarefas via terminal |
| `03-jokempo.py` | Jogo pedra, papel e tesoura |
| `04-caixa.py` | Simulação de caixa registradora |
| `05-dragao.py` | Jogo de aventura com dragão |
| `06-advinhador.py` | Jogo de adivinhação |
| `07-ranking-player.py` | Sistema de ranking de jogadores |
| `08-buscar_cotacoes.py` | Busca de cotações financeiras |
| `calculadora_modularizada/` | Calculadora com módulos separados |
| `Finance/` | Sistema bancário com POO (banco, conta, cliente) |
| `loja/` | Simulação de loja |
| `supermercado.py` | Simulação de supermercado |
| `water-tracker.py` | Rastreador de consumo de água |

### `03-MachineLearning/`
Estudos de ML com base no livro ISLP (Introduction to Statistical Learning with Python):
- `Anotacoes-ISLP.md` — anotações e resumos do livro
- `src/` — imagens e notebooks de apoio

### `04-Anotacoes/`
Anotações gerais de estudo:
- `DSA.md` — estruturas de dados e algoritmos
