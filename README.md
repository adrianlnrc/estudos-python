# Jornada de Estudos: Python

Este repositório foi criado para documentar minha evolução na linguagem Python, organizando o aprendizado entre conceitos teóricos e aplicações práticas baseadas em leituras e cursos.

# 📂 Estrutura do Repositório

- A organização segue uma lógica cronológica e temática:

- 01-teoria/: Notebooks Jupyter (.ipynb) com anotações e testes de novos conceitos, bibliotecas e sintaxes.

- 02-pratica/: Projetos e exercícios numerados em ordem de execução, desenvolvidos a partir de desafios de livros técnicos.

- referencias/ (Local): Material de apoio e cursos clonados para consulta (não versionados no GitHub).





## 🛠️ Tecnologias e Ferramentas

- Python 3.13.5

- Jupyter Notebook (para os arquivos de teoria)

- Git & GitHub (para versionamento)
